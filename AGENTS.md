# Notes for coding agents and automation

This repository is the **Dread.lol** Next.js app. When you change code:

1. Follow **[CONTRIBUTING.md](CONTRIBUTING.md)** — keep changes focused and match existing style.
2. Run **`npm run ci`** (typecheck, ESLint, Vitest) before suggesting a change is done; CI on `main` runs the same command.
3. **Security:** do not post secrets; see **[docs/SECURITY.md](docs/SECURITY.md)** and use private advisories for vulnerabilities.

**Cursor / editor rules** for this workspace live under **[`.cursor/rules/`](.cursor/rules/)** (for example, homepage feature updates and Notion tracking). Apply them when they match the task.

**Docs index:** **[docs/README.md](docs/README.md)**.
