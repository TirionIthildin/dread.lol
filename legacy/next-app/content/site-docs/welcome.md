## What is Dread.lol?

Dread.lol is a terminal-themed member profile site: dark grid background, scanlines, cyan accents, and JetBrains Mono. Members sign in (for example with Discord), customize a profile, and share a public URL.

## This documentation

These pages are written in Markdown and live in the repository under `content/site-docs/`. The sidebar and titles are configured in `lib/site-docs.ts`. To add a page, create a new `.md` file, register it in the nav, and deploy.
