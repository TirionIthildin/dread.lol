## Workflow

Work from the `main` branch and open pull requests against `main`. Keep changes focused and match existing patterns in the files you touch.

## Checks

CI runs typecheck, ESLint, and Vitest (`npm run ci`). Run the same command locally before opening a PR.

| Command | Purpose |
|--------|---------|
| `npm run test` | Unit tests |
| `npm run test:e2e` | Playwright E2E |
